A markup language ia a system for annotating a document in a way that is syntactically distinguishable form the text. It specifies the structure and presentation of text and other elements within a document.

The basic building blocks of HTML are elements, which are represent by tags. The tags are enclosed in a angle brackets(< >) and usually come in pairs, with an opening tag and a closing tag.

An HTML document typically consists of an <!DOCTYPE> declaration, an <html> element that contains a <head> and <body> section.

The <!DOCTYPE> declaration is used to specify the document type and version of HTML being used in the document. Its help the browser to render the web page correctly.

XHTML is a stricter and cleaner version of HTML that follows the rules of XML. It requires all tags to be properly nested, all attribure to be quoted, and all tags to be closed.

Empty elements in HTML are elements that do not have any content between their opening and closing tags. They are self-closing, and tag is optional.

The <meta> tag is used to provide metadata about the HTML document. It can include inform such as the character encoding, author, description, keywords, and viewport settings.

The <div> element is a block-level container used to group together other HTML elements and apply styles or scripts to them collectively.

The <span> element is an inline container used apply styles or scripts to a specific section of text or inline elements.

Semantic HTML is the used of HTML markup to reinforce the meaning of the information in web pages rather than merely define its presentation or look. It helps improve accessibility, search engine optimization, and the clarity of the code.

Both <strong> and <b> tags are used to make text bold. But <strong> is a semantic tag that indicates the enclosed text is of strong importance, while <b> is a presentation tag that simply applies bold styling.

Both <em> and <i> tags are used to italicize text, but <em> is a semantic tag that indicates emphasis, while <i> is a presentation tag that simply applies italic styling.

The <a> is used to create hyperlink. Allowing users to navigate between web pages or to different sections within the same page.

An absolute JRL contains the full address of the linked resource, include the protocol (http:// or https://),domain name, and path, while a relative URL specific the address of the linked resource relative to the current document.

The <img> tag is used to embed images into an HTML document.

HTML suports three types of lists: unordered list ( <ul> ), ordered lists( <ol> ), and difinition lists (sdl).

The <table> element is used to create tables, Which organize data into rows and columns.

The form element is used to create interactive forms for collecting user input, Which can then be submitted to a server for processing.

Form controls are elements used within a <form> element to collect user input, such as text inputs, checkboxes, radio buttons, select dropdowns, and buttons.

The GET method subkits form data as part of the URL in the query string, visible in the brower's address bar, while the POST method submits form data in the HTTP request boby, keeping it hidden form view.

The <input> element is used to create various types of form controls, such as text inputs, checkboxes, radio buttons and buttons.

The <textarea> element is used to create multi-line text input field within a form.

Semantic elements in HTML are tags that provide meaning to the content they enclose, such as <header>, <footer> , <nav>, <article>, <section>, <aside>, <main>.

The <header> element represents introductory content at the beginning of a page or section.
The <footer> element represents concluding content at the end of a page or section.

The <nav> element is used to define a section of navigation links within a document.

The <article> element is used to define independent, self-contained content that can be distributed and reused independently.

The <section> element is used to define a thematic grouping of content within a document, such as chapters, headers, footers, or any other grouping of content.

The <aside> element is used to define content that is tangentially related to the content around it, such as sidebar, pull quotes, or advertisements.

The <main> element is used to define the main content area of document, excluding any content that is repeated across multiple pages, such as headers, footers, or navigation links.

Data attributes are custom attributs that can be added to HTML elements to store extra information, they are prefixed with data and can be accessed using JavaScript.

The <iframe> element is used to embed another HTML document within the current document.

The <audio> and <video> elements are used to embed audio and video files, respectively, into an HTML document.

The <script> element is used to embed or reference JavaScript code within an HTML document.

When the async attribute is present, the script will be executed asynchronuously as soon as it is available, without blocking the rest of the page from loading without the async attribute, the script is executed synchronously, blocking the page load until it has finished downloading and executing.

Script with the defer attribute will not execute until the HTML document has been fully pased. where as scripts without the defer attribute will execute as soon as they are encountered in the document. potentially blocking rendering of subsequent content.

The alt attribute specifies an alternative text description for an image. which is displayed if the image fails to load or if the user is using a screen reader.

The title attribute provides an addtional information about an element, typically displayed as a tootlip when the user hovers over the element.

The href attribute specifies the URL of the linked resouce.
The target attribute specifies where to open the linked resource. such as in a new brower window or tab.

The rel attribute specifies the relationship between the current document and the linked resouce, such as stylesheet for linking to a stylesheet for indicating that the link should not be followed by search engines.

The src attribute specifies the URL of the external script file to be loaded by the <script> tag or the URL of the image to be displayed by the <img> tag.

The type attribute specify the MME type of the script being referenced, such as text/Javascript for JavaScript code.

The lang attribute specifies the character encoding used by the document such as UTF-8 or ISO-8859-1.

The lang attribute specifies the primary language of the content contained within the element, which helps assisitive technologies and search engies understand the content.

The role attribute specifies the role of the element in the document, specially for elements that do not have a semantic meaning or have an ambiguous meaning.

The aria-* attributes are used to enhance the accessibillity of HTML elements by providing additonal information to assistive technologies, such as screen readers.

The content editable attribute specifies whether the content of an element is editable by the user.

The hidden attribute specifies that an element is not yet, or is no longer, relevent to the documet's current state, and should be hidden from view.
