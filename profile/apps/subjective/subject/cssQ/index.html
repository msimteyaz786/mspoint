CSS Stand for Cascading Style Sheet. It is a language that is used to describe the style of a document and represent of a document written in HTML or XML. It allows web developers to control aspects like Layout, Colors, Fonts and Spacing of HTML elements.

Because CSS to control the layout, colors, fonts and overall visual appearance of web pages, making them more attractive and user friendly.

CSS can be applied in three ways:

1. Inline→ Directly within HTML element.

2. Internal stylesheet→ Within a <style> elementin the <head> section of an HTML document.

3. External stylesheet→ linked to an HTML document using the <link> element.

Inline styles are CSS styles applied Directly to an indiv Idual HTML element using the style attribute.

<p style="color:red; font-size:50px; face:calibri;">This is a paragraph with inline styles. </p>

→ An internal stylesheet is CSS code written within a <style> element in the <head> section of HTML document.

→ It applies styles to that particular HTML document only.

→ An external stylesheet is a separate CSS file linked to an HTML document using the <link> element.

→ It allows styles to be applied scross multiple HTML document.

You can link a CSS file to an HTML document using the <link> element within the <head> section of the HTML document.


<link rel="stylesheet" type="text/css" href="style.css">

A CSS selector is a pattern that is used to select and style HTML elements based on various criteria such as element type, Id, Classes attributes, and relationship with other elements.

There are four types of selector
1. Universal selector
2. Id selector
3. Element selector
4. Class selector

Universal Selector is select all elements on the page.

*{
   property: value;
}

Type Selector is selects all elements of a specific type.

p{
  property: value;
}

Class Selector is selects elements with a specific class attribute.

.classname{
  property: value;
}

Id Selector is selects elements with a specific Id attribute.

# Idname{
  property: value;
}

Attribute Selector is selects elements based on the presence or value of an attribute.

inuput[type="text"]{
  property: value;
}

Pseudo-classes is selects elements based on their special state.

a:hover{
  property: value;
}

Pseduo-element is selects specific parts of an elements content.

p::before{
  content: Prefix;
}

Descendant Selector selects elements that are descendant of another element.

div p{
  property: value;
}

Child Selector selects elements that are children of another element.

div > p{
  property: value;
}

Adjacent Sibling Selector selects the first elements that immediately follows a specified element, and both elements must have the same parent.

h1 + p{
   property: value;
}

General Sibling Selector selects element that are Sibling of a specied element.

h2 ~ p{
   property: value;
}
