C++ is an Object-Oriented Programming Language. It was developed by Bjarne Stroustrup in 1990.

Bjarne Stroustrup wants to develop a language that support object-oriented programming features with the powerful features of C language, and hence C++ evolved.

C++ consists of C language features along with class-object construct features of Simula67.

Initially C++ is named as C with Classes by Bjarne Stroustrup but in 1983, it was renamed to C++ (indicating an increment to C) proposed by Rrick Mascitti.

It is a superset of C language, while C++ contains features of C language and adds some of its own. Almost all programs written in C are also C++ programs.

C++ programming language provide many essential features including polymorphism, virtual and friend functions, templates, namespaces and operator overloading etc.

A programming language is an artifcial language which is used to desing instructions that are executed by a particular machine means 'computer' for performing some tasks means 'computation'.

POP stands for 'Procedure-oriented Programming language'. That is programming language in which a list of instructions is given to the computer to follow, in order to perform a task. These instructions are organized into groups which are known as functions(procedures). BASIC, BCPL, C COBOL etc all are Procedure-oriented programming language.

OOP stands for 'Object-oriented Programming language'. That is based on objects, instead of just functions and procedures. These objects are organized into classes, which allow individual objects to be group together. C++ C#, Java, Eiffel etc. all are Object-oriented programming language.

Source Code is a code which is written by the programming in human readable form with proper Programmign syntaxes.

Executable code is the machine understandable code. which can be executed by a machine Operating System.

Compiler is a software moduel which translate the source code of program into executable code.

Header file is a file which contains declaration of library functions, global variables (variables defined in other source files) and macro definitions. Extension of the header file is (.h).

Tokens individual units in a program are known as Token. Basically they are the individual words, symbol and punctuational marks.

Constants are the fixed values that do no change during execution run time of a program.

Variables jis a memory location in the program's memory which is used to store the data value or information. Values stored in a variables can be modified during program's execution.

Identifier is a tag name for a particular entity. which uniquely indetifies.

Datatype gives information about type-definitions, which is to be stored in a variable. Ex→ int, char, float, double, etc.

Keywords are the reserved word (predefine to the computer) whose meaning are already known to the Compiler. Ex→ int, char, if, else, return, void etc. There are total 32 Keywords in C language.

Operators is a mathematical symbol used to mainipulate between two data. In other word, Operators are the symbol which perform a predfined operation on operands.

The data of Operators are know as operands. Example→ 5+2, where 5 and 2 are operands and + is Operator.

Control or decision-making Statement are the statements in a program which can control the execution order of the statements, ex→ The order in which the instructions in a program must be executed.

Pointer is a special type of variable which can store the address an another variable.

Function or method are set of code that used to perform certain actions and in other word Function is a group of statements or instructions which perform a predfined task.

When a functions calls itself, it is known as Recursion.

Array is a collection of similar data type or a group of having similar data type.

Structure is the collection of variable having different data-type, under a single name tag.

Class is a blue-print fromo which objects are created. Class contains data members to stroe information and member function to operate upon data members.

Object is an instances of a class. It is basically a real-word implementation of the having all those property values which are defined or structured in the class.

Data Variables of a class are known as the data-members of the class.

Functions belonging to a particular class are known as member functions of that class.

Constructor is a special member function have same name as the class and used fot the automatic initialization of objects of the class. It gets of its class is created.

Destructor is a special member function having same name as the class name preceded by a tilde (~) and used to destroy the objects that are created by constructor.

The super keyword is used to invoke the overriden method. It also forwards a call from a constructor, to a constructor in the superclass.

This is a special constructor for creating a new object as a copy of an exixting object. There will be always only one copy constructor that can be either defined by the user or the system.

→ The base class is the most generalized class, and it is said to be a root of class.

→ A Subclass is a class that inherit from one or more base class.

→ The Superclass is the parent class from which another class inherits.

When a class inherits the property of another class, this is known as Inheritance.

Whose properties are inherited by another class that is called base class.

Derived class are those class, one who inherit the properties of the base class.

One base class and one derived class is called single Inheritance.

One base class and two or more than two derived classes that is called Hierarchical Inheritance.

One derived class and two or more that two base classes is called Multiple Inheritance.

One base class(A), One derived class(B) which in turn serves as a base class for one more derived(C) class that is called Multilevel Inheritance.

Combination of two one more inheritance is called Hybrid Inheritance.

An inline function is a technique used by the compiler and instructs to insert complete body of the function wherever that function is used in the program source code.

A virtual function is a member function of a class, and its functiolality can be overriden in its derived class. This function can be implemented by using a Keyword called virtual function.

A friend function is a friend of a class that is allowed to public, private, or protected data in that same class. If the function is defined outside the class cannot access such as any information.

Functions with same name having different number of arguments is called Function Overloading.

Functions with same name having same number of arguments is called Function Overriding.

'this' pointer is an object pointer which points to the currently calling object.

Friend Function is a function with the help of which we can access the private members of a class.

Abstract class is a class which cannot be instantianed.

Operator overloading is a way of providing new definitions for C++ operators.

Runtime errors are known as Exceptions.

Exception handling is a mechanism to handle runtime-error.

File handling is a way of dealing with the data on the secondary storage device (such as a hard disk) from a program. File handling operations include opening a file, reading a file, writing a file, closing a file etc.
